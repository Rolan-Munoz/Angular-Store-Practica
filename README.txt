En este ejercicio vamos a simular una tienda usando para ello el framework Angular para el Frontend, Mockoon para simular una API de backend, 
SCSS y bootstrap para darle estilos al proyecto.

1: Creamos los dos primeros componentes dentro de la carpeta layouts, los cuales seran el Navbar y el footer, y lo añadiremos a app.componets, 
pues seran dos partes fijas de la aplicacion.


2: Creamos el comoponente Home, que sera nuestra pagina principal. y el componente nosotros, que sera la pagina estatica de informacion sobre la  
tienda. Añadimos el enrutamiento necesario para ambos componentes.

3: Se han creado los dos componentes y su enrutamiento, dejamos el contenido y diseño de los mismos para una etapa posterior, pues en ellos 
habra informacion que nos proveera diferentes componentes y servicios que inyectaremos en ellos.
